Política de Privacidade

Esta Política de Privacidade descreve como as informações são tratadas ao utilizar o aplicativo Quanto deu?.

1. Coleta de informações

O aplicativo Quanto deu? não coleta dados pessoais sensíveis dos usuários, como nome, endereço, CPF, e-mail ou informações bancárias.

As informações inseridas pelo usuário no aplicativo, como valores, produtos consumidos e imagens adicionadas manualmente, são de responsabilidade do próprio usuário e são utilizadas apenas para o funcionamento do aplicativo.

2. Uso das informações

As informações inseridas no aplicativo são utilizadas exclusivamente para permitir o controle de consumo, cálculo de valores e visualização de histórico dentro do próprio aplicativo.

Nenhuma informação é vendida, compartilhada ou transferida para terceiros.

3. Compras no aplicativo

O aplicativo pode oferecer recursos pagos por meio de compras no aplicativo (In-App Purchases), processadas diretamente pela Google Play.

O desenvolvedor não tem acesso a dados financeiros, informações de pagamento ou dados do cartão do usuário.

4. Armazenamento de dados

Os dados criados no aplicativo são armazenados localmente no dispositivo do usuário ou conforme o funcionamento padrão da plataforma Android.

A desinstalação do aplicativo pode resultar na exclusão desses dados.

5. Permissões

O aplicativo pode solicitar permissões como acesso à câmera ou armazenamento, exclusivamente para permitir funcionalidades solicitadas pelo usuário, como adicionar imagens aos registros.

Nenhuma permissão é utilizada para coleta de dados sem o conhecimento do usuário.

6. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomenda-se que o usuário revise esta página para estar ciente de eventuais alterações.
